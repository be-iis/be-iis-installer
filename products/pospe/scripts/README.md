# Purpose
Installer scripts for the POSPE product variant. Used to automate setup and configuration on supported Raspberry Pi or Linux systems.

# Files
- `install.sh`: Executes the POSPE installation procedure. Intended to be run on the target system.

# Usage
Make `install.sh` executable and run as root:
```bash
chmod +x install.sh
sudo ./install.sh
```

# Notes
- Designed for use as part of the broader be-iis-installer project.
- Requires superuser privileges.
- Refer to the general project documentation for prerequisites and hardware details.

# Purpose
This directory is intended for device tree overlay files related to UART-capable hardware products, used to configure and enable UART interfaces on supported platforms (e.g., Raspberry Pi) during system boot.

Overlay files placed here are typically applied via platform-specific mechanisms (e.g., config.txt for Raspberry Pi), enabling or customizing UART hardware support. See higher-level documentation for platform integration steps.

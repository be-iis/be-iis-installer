# BE-IIS-HPP-MODBUS

## Isolated Dual RS-485 (Modbus) Industrial HAT++ for Raspberry Pi

Modbus over RS-485 enables robust long-distance multi-drop communication and is a well-established industrial communication standard.

The HAT provides two independent galvanically isolated RS-485 interfaces with automatic direction detection and is stackable within the BE-IIS HAT++ ecosystem.

---

## Key Features

- Dual RS-485 (Modbus) interface
- SC16IS752 dual I2C-to-UART controller
- Up to 1.15 MBit/s
- Galvanically isolated interfaces
- Configurable 120 Ω termination
- Raspberry Pi HAT+ compliant
- Stackable BE-IIS HAT++
- Configurable addressing and IRQ routing
- RoHS compliant

---

## Applications

- MODBUS evaluation
- Embedded prototyping
- Gateway development
- Industrial communication testing

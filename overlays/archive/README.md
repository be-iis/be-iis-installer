# Purpose
This directory is reserved for archived overlay files and related resources that are no longer actively used or maintained in the main overlay deployment. It serves as a storage area for historical or deprecated overlays and associated configurations.

# Usage
Do not place active or production overlay files in this directory. Reference overlays and files here only for legacy review or migration purposes. For active overlays, use the appropriate subdirectories under `overlays/`.

# Notes
- No overlays or scripts are present in this directory by default.
- Content, if any, may be reused or migrated manually as needed.

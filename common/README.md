# Purpose
Provides shared shell functions for scripts in the installer project.

# Files
- `lib.sh`: Collection of reusable shell functions for deployment and helper scripts.

# Usage
Source `lib.sh` in installer or setup scripts to access common functions:

```sh
source /path/to/common/lib.sh
```

# Notes
Only use within the context of this installer project. Functions require a Linux environment with standard utilities.

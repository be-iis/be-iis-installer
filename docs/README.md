# Purpose
This directory contains documentation relevant to setting up and flashing the Raspberry Pi OS for use with the BE-IIS Installer project.

# Files
- **be-iis_raspberry-pi-os-flashing-guide.md**: Instructions for flashing Raspberry Pi OS, including preparation steps and troubleshooting guidance.
- **README.md**: Overview of the documentation directory.

# Usage
Refer to `be-iis_raspberry-pi-os-flashing-guide.md` for step-by-step instructions on preparing the Raspberry Pi OS environment before installing BE-IIS components.

# Notes
This directory is for documentation only. No code or scripts are present here. Hardware setup and overlay configurations are described elsewhere in the repository.

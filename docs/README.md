# Documentation

This directory contains documentation relevant to setting up and flashing Raspberry Pi OS for use with the BE-IIS Installer project.

## Files

* **be-iis_raspberry-pi-os-flashing-guide.md**: Instructions for flashing Raspberry Pi OS, including preparation steps and troubleshooting guidance.
* **README.md**: Overview of the documentation directory.

## Usage

Refer to `be-iis_raspberry-pi-os-flashing-guide.md` for step-by-step instructions on preparing the Raspberry Pi OS environment before installing BE-IIS components.

## Additional Information

More technical background information, product details, and Linux software documentation are available on the BE-IIS website:

* Technology articles: https://www.be-iis.eu/technology/
* Linux software articles: https://www.be-iis.eu/linux-software/

These pages provide additional information about BE-IIS hardware, Single Pair Ethernet, Linux drivers, kernel modules, and practical setup examples.

## Notes

This directory is for documentation only. No code or scripts are present here.

Hardware setup, product-specific installation steps, and overlay configurations are described elsewhere in the repository.

# Purpose
Example configurations and setup instructions for field switch scenarios with BE-IIS hardware using Raspberry Pi.

# Files
- `network_setup_rsp2_bridge.md`: Network bridge setup guide for Raspberry Pi RSP2.
- `setup_rsp2.md`: Field switch configuration instructions for RSP2 hardware.
- `setup_rspz.md`: Field switch configuration instructions for RSPZ hardware.

# Usage
Consult the relevant setup file (`setup_rsp2.md` or `setup_rspz.md`) for detailed steps to configure the respective target hardware as a field switch. Use `network_setup_rsp2_bridge.md` for guidance on network configuration in bridge mode for RSP2 systems.

# Notes
- This directory does not contain executable scripts; only procedure documentation is provided.
- Ensure requirements described in the setup documents are met before applying configurations.
- Hardware model-specific details are covered in the respective setup files.
